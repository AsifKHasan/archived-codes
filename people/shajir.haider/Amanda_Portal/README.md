# Amanda-Portal

# Prerequisite

node js- 10.x
npm - 6.x

installing node js

for Linux and mac **sudo apt-get install nodejs**

installing npm 

for Linux and Mac **sudo apt-get install npm**

# For installing dependencies

**npm install**

# For running the DEV server

**ng serve**

# For Deployment

build the app first
**ng build --prod --aot**
